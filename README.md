# 🍳 Smart Cookbook

**Your AI-powered cooking assistant - 100% free, 100% private**

[![es](https://img.shields.io/badge/lang-es-yellow.svg)](README.es.md)

## ✨ Features

- 🤖 **Multiple AI providers**: OpenRouter, Groq, Google AI, Cerebras (all free!)
- 🔒 **100% Private**: Your data never leaves your browser
- 🌐 **No backend**: Everything runs client-side
- 🛡️ **Anti-Prompt Injection**: Protection against AI manipulation
- 🌙 **Dark/Light theme**: System, dark or light mode
- 🌍 **Multilingual**: Spanish and English
- 📱 **Mobile First**: Designed for mobile-first experience
- ⚡ **Ultra-fast**: Recipe generation in seconds

## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/yourusername/smart-cookbook.git
cd smart-cookbook

# Install dependencies
npm install

# Run in development
npm run dev

# Open http://localhost:5551
```

## 🛡️ Security

### Anti-Prompt Injection Protection

The system includes multiple layers of defense:

1. **Input validation**: Blocks patterns like "ignore previous", "you are now", etc.
2. **Hardened system prompt**: Immutable instructions that AI cannot ignore
3. **Output validation**: Verifies that the response is culinary content

### Privacy

- ✅ **No proprietary backend**: Everything is client-side
- ✅ **No analytics/tracking**: We don't collect data
- ✅ **No third-party cookies**: Only localStorage
- ✅ **Secure API keys**: Only sent to the provider you choose
- ✅ **"Delete My Data" button**: Erases everything instantly
